
Ce projet est une **API REST simple** réalisée avec **Node.js et Express**.  
Il permet de gérer une liste d'utilisateurs via un fichier `users.json`.

---

## Structure du projet
TP1_NodeJS/
├── app.js # Serveur Express principal
├── users.json # Fichier JSON contenant les utilisateurs
├── public/
│ └── index.html
└── README.md
