Bienvenue sur l'API Express de démonstration

Essayez les routes suivantes :

  • /api/users - Tous les utilisateurs
  • /api/users/1 - Utilisateur avec ID 1
  • /api/users/2 - Utilisateur avec ID 2
  • /api/users/3 - Utilisateur avec ID 3

Vous pouvez également tester la création d'un nouvel utilisateur avec une requête POST sur /api/users via Postman ou Thunder Client.

Exemple JSON pour POST :

{
  "nom": "Ettouyjer",
  "prenom": "Yasmine",
  "age": 21
}
